== Description ==

**Cur8** transforms your WordPress site into a beautiful content curation platform. Share status updates, images, quotes, and locations through an elegant web interface at `/wptext`.

* Shortcode: `[cur8_updates]`

= Web Interface =

Access the posting interface at `yoursite.com/wptext` to quickly create updates without accessing the WordPress admin.

= Customization =

* Choose from 5 design themes (Modern, Minimalist, Bold, Shadow, Gradient)
* Customize primary colors
* Adjust layout and styling
* Filter by update type

== Installation ==

1. Upload the `cur8` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit Settings ‚Üí Cur8 to customize colors and themes
4. Use the shortcode `[cur8_updates]` to display updates
5. Access `/wptext` to create new updates

== Frequently Asked Questions ==

= How do I display updates on my site? =

Use the shortcode `[cur8_updates]` in any post or page, or add the Cur8 widget to your sidebar.

= Can I customize the appearance? =

Yes! Go to Settings ‚Üí Cur8 to choose from multiple design themes and customize colors.

= Where is the posting interface? =

Navigate to `yoursite.com/wptext` to access the beautiful posting interface.

= Can I add links to status updates? =

Yes! Status updates support optional links with custom link text.

= Is it mobile-friendly? =

Absolutely! Cur8 is fully responsive and works beautifully on all devices.
